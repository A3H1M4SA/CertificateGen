Introduction

This University Project, part of Anglia Ruskin courses, showcases an integration between Java and PHP, focusing on creating a scalable certificate generator application. Utilizing OOP principles, the project emphasizes modularity, reusability, and maintainability in software development. The Java project uses a PHP API developed by me to generate the certificates.

Customization and Flexibility

The application provides extensive customization options for certificates:

  • Add new certificates and assign specific fonts for each certificate.
  • Assign specific fonts for each field (Name, Sign, Date).
  • Assign any color for the fonts of any field.
  • Add fields and change any field location by X/Y coordinates.
  • When certificates are saved, they are formatted in an A4 cuttable format, making it easy to cut them out after printing.

All these customizations must be made on the PHP side, and the application is ready to use as the API is hosted for demonstration purposes.

Usage Instructions

  1. Upon running the application, you will be greeted with a welcome screen.
  2. Click the 'Continue' button to proceed.
  3. Click on 'Choose Path To Save Certificate' to open a file explorer window. Select the desired path where the certificate will be saved.
  4. After selecting the path, click 'Next'.
  5. In the certificate window, choose a certificate style from the dropdown menu.
  6. Fill in the required fields to customize the certificate.
  7. Click 'Preview' to see an updated preview of the certificate.
  8. If satisfied, click 'Save' to save the certificate.

Database and API Configuration

Details of the database fields used for certificate customization:

  • certificate_id: Unique identifier for each certificate.
  • name_loc, signedby_loc, company_loc, date_loc: X/Y coordinates for placeholders.
  • name_textcolor, signedby_textcolor, company_textcolor, date_textcolor: RGB color codes for text.
  • name_fontsize, signedby_fontsize, company_fontsize, date_fontsize: Font sizes for each text field.
  • primaryfont: Primary font used for Name and Company Name.
  • certificate_name: Name of the certificate style, e.g., abcd.
  • certificate_path: Path to the certificate image file, e.g., assets/abcd.png.
  • certificate_used: Counter to track how many times each certificate is used.

Self-Hosting Instructions

  1. For self-hosting, upload the PHP files (dbconnect.php, api.php, and preview.php) to your server.
  2. Import the provided SQL file into your database to set up the necessary tables.
  3. Adjust the API_BASE_URL in the Java source file to point to your new server location.
