Introduction

This University Project, part of Anglia Ruskin courses, showcases an integration between Java and PHP, focusing on creating a scalable certificate generator application. Utilizing OOP principles, the project emphasizes modularity, reusability, and maintainability in software development.

Prerequisites

  • Java JDK 11 or higher
  • IntelliJ IDEA or any compatible Java IDE installed

Installation and Setup

  1. Open IntelliJ IDEA.
  2. Import the Certificate Generator project.

Running the Application

  1. Ensure that the Java JDK is correctly installed on your system by running `java -version` in your terminal.
  2. Launch the project in IntelliJ IDEA.
  3. Run the application by clicking on the run button.

Usage Instructions

  1. Upon running the application, you will be greeted with a welcome screen.
  2. Click the 'Continue' button to proceed.
  3. Click on 'Choose Path To Save Certificate' to open a file explorer window. Select the desired path where the certificate will be saved.
  4. After selecting the path, click 'Next'.
  5. In the certificate window, choose a certificate style from the dropdown menu.
  6. Fill in the required fields to customize the certificate.
  7. Click 'Preview' to see an updated preview of the certificate.
  8. If satisfied, click 'Save' to save the certificate.
